# Stock Compass

휴대폰에서 설치형 앱처럼 쓸 수 있는 주가 전망 보조 PWA입니다. 일반 사용자가 전문용어를 몰라도 읽을 수 있도록 쉬운 문장 중심으로 구성했습니다.

## 포함된 기능

- 종목 코드 입력 후 실제 일별 가격 자동 연결
- 다음 거래일 방향성 확률 계산
- 추세, 최근 흐름, 과열 가능성, 가격 흔들림, 낙폭을 쉬운 문장으로 표시
- 리스크와 간단한 사후 검증 탭
- 관심종목 저장
- 관심종목별 알림 조건 화면
- 오프라인 캐시와 앱 설치 매니페스트

## 계산 방식

단일 지표 하나로 찍는 방식이 아니라, 학술/실무 퀀트 분석에서 널리 쓰이는 요소를 합칩니다.

- 로그수익률 기반 단기 추세
- 최근 한 달 평균 대비 위치
- 3개월 평균 대비 중기 추세
- 단기 과열/침체 점검
- 최근 변동성 및 고점 대비 낙폭
- 과거 데이터를 한 칸씩 가려가며 검증하는 워크포워드 방식

예측은 확률적 판단이며 수익이나 특정 방향을 보장하지 않습니다.

## 실행

로컬에서 실제 데이터 자동 연결까지 쓰려면 아래 파일을 실행하세요.

`Stock-Compass-실행.cmd`

실행하면 서버가 켜지고 브라우저가 아래 주소로 열립니다.

`http://127.0.0.1:4174/`

## 출시 구조

Vercel 같은 서버리스 호스팅에 이 폴더를 올리면 `/api/prices?symbol=NVDA`가 실제 가격 데이터를 가져오고, 화면은 그 API만 호출합니다. 사용자는 API 키나 데이터 연결을 볼 일이 없습니다.

## 주의

이 앱은 투자 판단을 돕는 분석 도구입니다. 실제 출시 버전에서는 정식 라이선스가 있는 금융 데이터 공급자와 서버 API를 붙이는 것이 좋습니다.
